# OpenChapter
Sistema de gerenciamento de biblioteca fullstack completo com Java SpringBoot, Thymeleaf e MongoDB Atlas.

### Tecnologias: 
Java 17 com Spring Boot| 
MongoDB (Banco de dados NoSQL)| 
Thymeleaf (Templates)|
Spring Security (Autenticação)|
Lombok (Produtividade)

### Funcionalidades
Livros: Cadastro e consulta.|
Empréstimos: Controle de retiradas.|
Reservas: Sistema com status (Ativa/Cancelada) e prazo de 7 dias.|
Relatórios: Dashboard com estatísticas do acervo em tempo real.

### Como rodar
Clone o projeto.|
Certifique-se de que o MongoDB está ativo na porta 27017.|
Execute o comando ./mvnw spring-boot:run.|
Acesse localhost:8080.
